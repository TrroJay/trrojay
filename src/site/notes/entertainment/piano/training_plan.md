# 1 训练计划
## 1.1 乐曲的歌唱性
- 舒伯特的 D.899 是浪漫主义时期早期的作品，其歌唱性极强。其歌唱性可以体现在几个方面中：
	- 气息
		- 附点音符的使用代表了歌唱中的呼吸与节奏
		- 每个乐句呈现出“枣核”的结构，在乐句的末尾留出呼吸的气口
	- 音域
		- 旋律织体集中在中高音区，与歌唱时人声的旋律位置类似
		- 左手伴奏起烘托作用
## 1.2 音阶跑动
### 1.2.1 Analysis
- 好的音阶有什么特点
	- 每个音清晰，明亮，等时值；跑动快速流畅，匀速前进；以四个音或八个音为一组，掌握轻重力度；左右手下键一致。
- 我的音阶
	- 左右手弹奏无法做到一致。
		- 使用单手弹奏，另一只手打节拍的方式，节拍手进入 auto pilot 的状态，这种情况下大脑节拍器用于指挥弹奏手, 需要节拍手自动跟上。
		- 不过度依赖节拍器，只在几组后进行一次对齐校正操作。
	- 在每一组的八个音中容易突然出现重音，有时会出现附点结构。
		- 重音容易出现在大指上，这是由于大指在穿指后手掌需要借力进行平移，慢练时锻炼大鱼际肌肉群即可。
		- 出现附点结构通常是由于手腕移动没有跟上，可以放慢速度，提升耳朵分辨哪一个音时值过长的能力。
	- 跑动时手腕僵硬。
		- 多进行大幅度手腕移位，在弹奏重音后放松手腕，不应当紧绷着。

# 3 Musical Expression
## 3.1 Speed & Volumn
- 渐慢
	- r.i.t (Ritardando)
		- 节奏渐慢，不一定渐弱。用于表示乐段的转折。
	- rall. (Rallentando)
		- 渐慢，渐弱。用于乐句大结尾，表示情感的宣泄。
## 3.2 Touch
- stac. (staccato)
- legato
- marc. (Marcato [marˈkaːto])
	- **独立性：** 每个音符都应该是独立的，不应该含糊不清地连在一起。
	- **强调感：** 每一个音符都要像被“盖了章”一样，有明显的起始感。
- ten. (Tenuto [teˈnuːto])
	-  **时值上的“满”：** 必须弹满音符的全部时值，甚至要感觉到音符微微“溢出”了一点。绝不能提前松手。
	- **触键上的“重”：** 这不是指大力的敲击，而是一种**深沉的压力**。想象你的手指像吸盘一样吸在琴键上，把手臂的重量沉到底。推入。
