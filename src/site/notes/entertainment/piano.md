# 1 训练计划
## 1.1 乐曲的歌唱性
- 舒伯特的 D.899 是浪漫主义时期早期的作品，其歌唱性极强。其歌唱性可以体现在几个方面中：
	- 气息
		- 附点音符的使用代表了歌唱中的呼吸与节奏
		- 每个乐句呈现出“枣核”的结构，在乐句的末尾留出呼吸的气口
	- 音域
		- 旋律织体集中在中高音区，与歌唱时人声的旋律位置类似
		- 左手伴奏起烘托作用
## 1.2 音阶跑动
### 1.2.1 Analysis
- 好的音阶有什么特点
	- 每个音清晰，明亮，等时值；跑动快速流畅，匀速前进；以四个音或八个音为一组，掌握轻重力度；左右手下键一致。
- 我的音阶
	- 左右手弹奏无法做到一致。
		- 使用单手弹奏，另一只手打节拍的方式，节拍手进入 auto pilot 的状态，这种情况下大脑节拍器用于指挥弹奏手, 需要节拍手自动跟上。
		- 不过度依赖节拍器，只在几组后进行一次对齐校正操作。
	- 在每一组的八个音中容易突然出现重音，有时会出现附点结构。
		- 重音容易出现在大指上，这是由于大指在穿指后手掌需要借力进行平移，慢练时锻炼大鱼际肌肉群即可。
		- 出现附点结构通常是由于手腕移动没有跟上，可以放慢速度，提升耳朵分辨哪一个音时值过长的能力。
	- 跑动时手腕僵硬。
		- 多进行大幅度手腕移位，在弹奏重音后放松手腕，不应当紧绷着。

# 4 流行曲
### 4.1.1 梦中的婚礼 Mariage d'Amour
- 【【原创经典】理查德·克莱德曼为你演奏《梦中的婚礼》】 https://www.bilibili.com/video/BV15Y41117Y3/?share_source=copy_web&vd_source=2d5d054ab74f94c10a195a08b7a2a7d2
	- Richard 的左手富有变化，虽然全曲都是 legato 三连音，但开头的三连音的第三个音几乎完全隐藏（Pianissimo），起到了垫音的效果（保存了高音部分织体纯粹），高潮部分三连音节奏感强，多个三连音成组俨然已经构成了一个左手乐句，烘托着氛围。
	- Richard 的右手触键方式与众多普通演奏者不同，他使用的触键方式形成铃声般的清脆感。
